## ⚡ Quick Deploy (3 Commands)

```bash
cd Frontend
yarn build
npx netlify-cli deploy --prod --dir=out
```

**Done! 🚀** Your app is now live and installable!

## 🎨 Customize (Optional)

### Change App Name
Edit: `public/manifest.json`
```json
"name": "Your App Name"
```

### Change Icons
```bash
# Replace public/logo.png with your logo
yarn generate-icons
yarn build
```

### Change Colors
Edit: `public/manifest.json`
```json
"theme_color": "#your-color"
```
