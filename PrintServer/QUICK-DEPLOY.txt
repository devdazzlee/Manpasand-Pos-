================================================================================
QUICK DEPLOY GUIDE - Print Server to New Laptop
================================================================================

FILES TO COPY:
--------------
Copy the ENTIRE PrintServer folder to the new laptop

STEPS ON NEW LAPTOP:
--------------------
1. Install Node.js (if not installed)
   Download from: https://nodejs.org/

2. Copy PrintServer folder to new laptop
   Example: D:\Konain Bhai\Manpasand-Pos-\PrintServer

3. Open Command Prompt in PrintServer folder
   Run: npm install

4. Right-click: install-and-start.bat
   Select: "Run as administrator"
   Wait for completion

5. Done! Service will start automatically on boot

VERIFICATION:
------------
- Open browser: http://localhost:3001/health
- Should see: {"status":"ok",...}

- Check Services (services.msc):
  Find "Manpasand Print Server"
  Should show: Running, Startup type: Automatic

================================================================================
FILES TO RUN (In Order):
================================================================================

PRIMARY SETUP (Run This First):
-------------------------------
1. install-and-start.bat
   → Right-click → Run as administrator
   → Does everything automatically

IF PRIMARY DOESN'T WORK:
-------------------------
2. reinstall-service-fixed.ps1
   → Right-click → Run with PowerShell (as Administrator)
   → Alternative installation method

VERIFICATION/FIX (Optional):
----------------------------
3. ensure-service-auto-start.ps1
   → Right-click → Run with PowerShell (as Administrator)
   → Verifies and fixes automatic startup configuration

================================================================================
SUMMARY:
================================================================================
Just run: install-and-start.bat as Administrator
That's it! The service will be installed and configured for automatic startup.
