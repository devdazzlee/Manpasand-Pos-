================================================================================
SIMPLE SETUP FOR NEW LAPTOP (Using PowerShell Files)
================================================================================

Since .bat files don't work on your laptop, use these PowerShell files instead.

================================================================================
STEP-BY-STEP INSTRUCTIONS:
================================================================================

STEP 1: Copy PrintServer Folder
--------------------------------
Copy the ENTIRE PrintServer folder to the new laptop

STEP 2: Install Node.js (if not installed)
--------------------------------------------
Download from: https://nodejs.org/
Install it

STEP 3: Install Dependencies
----------------------------
1. Open PowerShell in PrintServer folder
2. Run: npm install
3. Wait for completion

STEP 4: Install the Service (MAIN STEP)
----------------------------------------
Right-click this file:
  → reinstall-service-fixed.ps1
  → Select "Run with PowerShell"
  → Select "Run as Administrator" (if prompted)
  → Wait for completion

STEP 5: Verify Setup (Optional)
---------------------------------
Right-click this file:
  → ensure-service-auto-start.ps1
  → Select "Run with PowerShell"
  → Select "Run as Administrator" (if prompted)
  → This verifies everything is configured correctly

================================================================================
FILES TO RUN ON NEW LAPTOP:
================================================================================

PRIMARY FILE (Run This):
------------------------
1. reinstall-service-fixed.ps1
   → Right-click → Run with PowerShell (as Administrator)
   → This does everything: installs service, sets to auto-start, starts it

VERIFICATION FILE (Optional):
------------------------------
2. ensure-service-auto-start.ps1
   → Right-click → Run with PowerShell (as Administrator)
   → Verifies and fixes automatic startup configuration

================================================================================
QUICK REFERENCE:
================================================================================

On New Laptop:
1. Copy PrintServer folder
2. Run: npm install
3. Right-click: reinstall-service-fixed.ps1 → Run with PowerShell (Admin)
4. Done!

The service will:
- Install automatically
- Set to start automatically on boot
- Start now
- Work without customer interaction
