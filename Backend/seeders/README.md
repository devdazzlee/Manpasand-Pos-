## Running Seeders

### Run seeders:
```bash
# Seed users
npm run seed:users

# Seed Dried Fruits & Nuts products
npm run seed:dried-fruits-nuts

# Seed Spices products
npm run seed:spices

# Seed Dates products
npm run seed:dates

# Seed Herbs products
npm run seed:herbs

# Seed Traditional Confectionery products (Fast Mode)
npm run seed:traditional-confectionery

# Seed Grains, Pulses & Rice products (Fast Mode)
npm run seed:grains-pulses-rice

# Seed Arqiat & Juices products (Fast Mode)
npm run seed:arqiat-juices

# Seed Flour products (Fast Mode)
npm run seed:flour

# Seed Crackers products (Fast Mode)
npm run seed:crackers

# Seed Pickles, Jams & Honey products (Fast Mode)
npm run seed:pickles-jams-honey

# Seed Irani Products (Fast Mode)
npm run seed:irani-products

# Seed Indian Products (Fast Mode)
npm run seed:indian-products

# Seed Nimco products (Fast Mode)
npm run seed:nimco

# Seed Essential Oils & Shampoo products (Fast Mode)
npm run seed:essential-oils-shampoo

# Seed SCENT & PERFUMES products (Fast Mode)
npm run seed:scent-perfumes

# Seed General products (Fast Mode)
npm run seed:general
```

### Run individual seeder directly:
```bash
npx ts-node seeders/seed-users.ts
npx ts-node seeders/seed-dried-fruits-nuts.ts
npx ts-node seeders/seed-spices.ts
npx ts-node seeders/seed-dates.ts
npx ts-node seeders/seed-herbs.ts
npx ts-node seeders/seed-traditional-confectionery.ts
npx ts-node seeders/seed-grains-pulses-rice.ts
npx ts-node seeders/seed-arqiat-juices.ts
npx ts-node seeders/seed-flour.ts
npx ts-node seeders/seed-crackers.ts
npx ts-node seeders/seed-pickles-jams-honey.ts
npx ts-node seeders/seed-irani-products.ts
npx ts-node seeders/seed-indian-products.ts
npx ts-node seeders/seed-nimco.ts
npx ts-node seeders/seed-essential-oils-shampoo.ts
npx ts-node seeders/seed-scent-perfumes.ts
npx ts-node seeders/seed-general.ts
```

## Notes

- Seeders are **idempotent** - safe to run multiple times
- Existing records will be skipped (no duplicates created)
- Passwords are automatically hashed using bcrypt
- Branches are created automatically if they don't exist

## Adding New Seeders

1. Create a new `.ts` file in this folder
2. Follow the pattern from `seed-users.ts`
3. Add a script to `package.json` if needed:
   ```json
   "seed:your-seeder": "ts-node seeders/your-seeder.ts"
   ```
